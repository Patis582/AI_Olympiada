# Základní řešení (baseline), které máte překonat

Baseline je jednoduchý **pravidlový klasifikátor**, který se rozhoduje **pouze podle
prezentace** pacienta – tedy podle vitálních funkcí a symptomů, které vidí každý.

Zhruba funguje takto:
- kritické vitály (nízká saturace, nízký tlak, velmi vysoká/nízká tepová frekvence, porucha
  vědomí) → nejvyšší priorita,
- středně odchýlené vitály nebo zjevně závažný symptom → vyšší priorita,
- mírné příznaky při stabilních vitálech → nižší priorita,
- jinak nízká priorita.

**Co baseline NEdělá:** nedívá se do anamnézy ani na epidemiologickou situaci. Tam je váš
prostor – ukažte, že díky tomu zachytíte případy, které baseline mine, aniž byste přitom
začali zbytečně eskalovat i lehké pacienty.

Vaším cílem je na stejné frontě pacientů dosáhnout **nižšího podhodnocení rizikových** než
baseline – a to číslo ukázat.
